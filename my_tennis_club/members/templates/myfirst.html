¿Que es Django?


Django es un framework de Python que facilita la creacion de sitios web utilizando Python.



¿Como funciona Django?


Cuando haya instalado Django y creado su primera aplicación web Django, y el navegador solicite la URL, esto es básicamente lo que sucede:


  • Django recibe la URL, verifica las urls en urls.py y llama a la vista que coincide con la URL.
  • La vista, ubicada en views.py, busca modelos relevantes.
  • Los modelos se importan desde el models.pyarchivo.
  • Luego, la vista envía los datos a una plantilla especificada en la templatecarpeta.
  • La plantilla contiene etiquetas HTML y Django, y con los datos devuelve el contenido HTML terminado al navegador.

    • Estructura de Django


    Django sigue el patrón de diseño MVT (Modelo Vista Plantilla).


  • Modelo: Los datos que desea presentar, generalmente datos de una base de datos.
  • Vista: un controlador de solicitudes que devuelve la plantilla y el contenido relevantes, según la solicitud del usuario.
  • Plantilla: un archivo de texto (como un archivo HTML) que contiene el diseño de la página web, con lógica sobre cómo mostrar los datos.


    • Ventajas


  • Desarrollo rapido
  • Interfaz de administrador incorporada
  • Soporte para ORM y bases de datos
  • Escalabilidad y estabilidad


    • Desventajas


  • Gastos generales para proyectos pequeños
  • Preocupaciones sobre el rendimiento
  • Desafio de la migracion
